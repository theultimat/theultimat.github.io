About

⤒

Hi, I'm Matt. I have professional experience in the areas of both digital subscriber line and machine learning, primarily working with C, C++, Python, and assembly on multi-core embedded platforms. I'm currently working as a Firmware Engineer at Broadcom.

I studied Computer Science at the University of Bristol where I completed my MEng in 2017. My final dissertation project explored real-time hair simulation and rendering on the GPU.

RISC-V Processor Simulator

While studying at university we were tasked with creating a cycle-accurate processor simulator for the Advanced Computer Architecture unit. This project is an evolution of the original assignment, written in C++ and implementing the RISC-V ISA with support for RV32I. This is an ongoing project - more information and source code will be available soon!

Raspberry Pi Laser Tag

This is an embedded project written in C making use of Raspberry Pi Pico boards and infrared LEDs to create a game of laser tag. This is an ongoing project - more details will be available soon!

Raytracer

Evolving Neural Networks
